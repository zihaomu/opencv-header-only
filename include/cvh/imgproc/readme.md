这个文件夹包含主要的opencv 图像处理函数。
分一下几个阶段实现：

阶段一：基础算力与色彩转换 (Core 运算 + Imgproc 基础)开始让像素动起来，实现逐像素的快速操作。
算术与逻辑运算： 矩阵的加、减、乘、除、按位与/或/非。
色彩空间转换 (cvtColor)： 实现 BGR $\leftrightarrow$ RGB、BGR $\leftrightarrow$ Gray。
这些是查表法（LUT）和简单乘加运算的好地方。

阶段二：空间几何变换 (Imgproc 几何)进入涉及坐标映射的算法。
插值算法： 核心是写出高效的最近邻插值（Nearest）和双线性插值（Bilinear）。
几何变换： * resize (缩放)warpAffine (仿射变换：平移、旋转、倾斜)warpPerspective (透视变换：四点映射投影)

阶段三：邻域操作与滤波 (Imgproc 滤波)涉及矩阵卷积运算。
卷积引擎： 编写一个通用的 2D 卷积函数（注意处理边界 Padding 问题）。
常用滤波器： 基于通用卷积实现均值滤波 (blur)、高斯滤波 (GaussianBlur)。
非线性滤波： 中值滤波（排序算法的优化）、双边滤波（涉及空间距离和像素差值的权重计算）。

阶段四：结构提取 (Imgproc 进阶)开始让库具备“视觉感知”能力。
边缘检测： Sobel 算子计算梯度，然后实现 Canny 算法（包含非极大值抑制和双阈值边缘连接）。
形态学操作： 膨胀 (Dilate) 和 腐蚀 (Erode)。直线检测： 霍夫变换 (Hough Lines)，这需要一定的极坐标数学基础。

阶段五：高级数学与频域分析 (Core 终极挑战)这是最难啃的骨头，建议放在最后。
一维与二维离散傅里叶变换 (DFT/FFT)： 用于频域滤波。
奇异值分解 (SVD) 与 PCA： 用于降维和特征分析。
这里如果手写太难，可以考虑引入类似 Eigen 的超轻量级 header-only 矩阵库作为可选依赖。
