# `example` 目录规划

## 目录职责

提供可直接运行的最小示例，验证“从 `cv::` 迁移到 `cvh::` 成本可控”。

## 示例分层

- `core` 示例：`Mat` 创建、类型转换、基础算子。
- `imgproc` 示例：颜色转换、resize、滤波。
- `imgcodecs` 示例：读图 -> 处理 -> 写图。
- `highgui` 示例：`imshow + waitKey` 弹窗验证。

## 当前可运行示例

- `highgui_imshow.cpp`：
  - 传入图片路径：读取并弹窗显示。
  - 不传参：自动生成测试图并弹窗显示。
  - 可选第二参数 `wait_ms`：控制 `waitKey(wait_ms)`（`0` 为一直等待）。

## 阶段计划

### P1：最小可用示例

- 每个模块至少 1 个端到端示例。
- 示例代码只依赖公开头，不访问内部路径。

### P2：迁移导向示例

- 增加“OpenCV 等价写法”对照片段。
- 记录当前不兼容点和替代写法。

### P3：性能导向示例（可选）

- 增加线程与 SIMD 开关示例，帮助用户选择运行模式。

## 完成定义（DoD）

- 示例可在 CI 编译运行。
- 示例覆盖总规划中的 MVP API。
